# Saint Lawrence University Institute Ndop - Smart Campus ERP

This rebuild expands the SLUI management system into a more connected academic and identity platform.

## Major additions in this version
- Bulk enrollment with tick-box selection for multiple students from the same program and semester into multiple courses
- Bulk attendance entry for whole classes
- Bulk assessment ledger entry for continuous assessment
- Bulk exam ledger entry for examination marks
- Admission number format updated to `SLUYYXXX` such as `SLU26001`
- Student accounts generated in bulk or one by one
- Student profile photo upload inside the student portal
- 3D-style student ID card with student photo and barcode
- Admin-uploaded role-based background images for admin, staff, student and registrar interfaces
- Uploaded school logo used across the platform
- Editable settings for institution branding, transcript labels, prefixes and account defaults

## Notes
- This is a browser-based app using local storage on the device/browser where it is opened.
- PDF export works best while online because the PDF libraries are loaded from a CDN.
- Student photos uploaded from student accounts are reflected on the 3D ID card automatically.
